## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/lorenzopalaia/MediaShift.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Build the project
   ```sh
   npm run build
   ```

<p align="right">(<a href="#readme-top">back to top</a>)</p>

## Usage

1. Run
   ```sh
   npm run start
   ```
